## 🛠️ Tech Stack

- **Frontend:** React, Tailwind CSS  
- **Backend:** Node.js, Express  
- **Database:** MongoDB  

---

## 🚀 Features

- User authentication and order placement  
- Admin dashboard with inventory tracking
